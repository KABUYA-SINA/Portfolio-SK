# 💼 Portfolio — Kabuya Sina

Portfolio personnel présentant mes projets, compétences et expériences en développement web.

---

## 🚀 Description

Ce portfolio a été conçu pour présenter mes différents projets en développement web ainsi que mes compétences techniques.
Il sert de vitrine professionnelle afin de mettre en avant mon parcours, mes réalisations et mon évolution dans le domaine du développement.

---

## 🎯 Objectifs du projet

* Présenter mes projets de manière claire et structurée
* Mettre en avant mes compétences techniques
* Fournir un accès rapide à mes réalisations et profils professionnels
* Créer une interface moderne et responsive
* Développer un site personnel évolutif

---

## 🛠️ Technologies utilisées

* React.js
* JavaScript
* SCSS
* SASS
* GitHub Pages (déploiement)

---

## ⚙️ Fonctionnalités

* 🧭 Navigation fluide entre les sections
* 📁 Présentation des projets
* 🧑‍💻 Section compétences
* 📱 Design responsive (mobile / desktop)
* 🔗 Liens vers GitHub et projets en ligne

---

## 🧠 Compétences développées

* Développement front-end avec React
* Structuration d’une application web complète
* UI/UX design
* Déploiement via GitHub Pages
* Organisation de projets personnels
